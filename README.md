## 구현 기능

### 기능 1
  서버와의 통신을 통한 로그인 기능 > 만일 이전에 회원가입을 했다면 JWT를 로컬에 저장하므로서 회원가입 또는 로그인을 하지 않아도 바로 이동 가능!
### 기능 2
  Todo 기록 및 달성률(추가 예정)
### 기능 3
  목표 달성시 취소선을 그어 좀더 목표달성 유도
